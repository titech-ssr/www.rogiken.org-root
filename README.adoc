== About
このリポジトリは link:https://www.rogiken.org/[東京科学大学 ロボット技術研究会] のウェブサイトのファイルを管理するものである。

ファイル群へのコンパイル等は一切行われず、リポジトリ上のファイルがそのままウェブサイトとして配置される。

また、動的なコンテンツや変換を必要とするファイルを含むような特定のサブディレクトリ(たとえばblogやwiki等)は、このリポジトリでは管理しない。
